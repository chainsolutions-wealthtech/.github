# ChainSolutions WealthTech

Organisation GitHub cible pour les projets WealthTech, BRVM, OPCVM, MCP, automatisation, migration et gouvernance technique.

## Rôle de cette organisation

Cette organisation sert de cible principale pour :

- les dépôts applicatifs WealthTech ;
- le serveur MCP ;
- les projets BRVM / OPCVM / finance de marché ;
- les migrations depuis les comptes sources ;
- la documentation technique ;
- les workflows de gouvernance ;
- les Pull Requests contrôlées.

## Règle MCP

Le MCP doit utiliser `chainsolutions-wealthtech` comme organisation cible principale.

Les comptes sources actuellement connus sont :

- `Patricked-code`
- `Wealthtechinnovations`

Les nouveaux dépôts doivent être créés ici, sauf exception documentée.

# Contribution

Règles de contribution pour `chainsolutions-wealthtech` :

1. Lire la documentation du projet avant modification.
2. Créer une branche dédiée.
3. Éviter toute régression.
4. Documenter les changements.
5. Tester avant Pull Request.
6. Ne jamais exposer de secret.
7. Mettre à jour `SUIVI.md`, `CHANGELOG.md` ou la mémoire projet si nécessaire.

Organisation cible par défaut : `chainsolutions-wealthtech`.
